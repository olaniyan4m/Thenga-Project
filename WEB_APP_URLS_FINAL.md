## 🔧 **Manual Deployment Steps**

Since the automated deployment is having network issues, here's how to deploy manually:

### **Step 1: Deploy Business App**
```bash
cd standalone-pwa
firebase deploy --only hosting
```

### **Step 2: Deploy Customer App**
```bash
cd apps/pwa
firebase deploy --only hosting
```
