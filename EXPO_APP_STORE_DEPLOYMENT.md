## 🚀 **Deploy to App Store (3 Commands):**

### **Step 1: Install EAS CLI**
```bash
npm install -g @expo/eas-cli
```

### **Step 2: Configure EAS**
```bash
eas build:configure
```

### **Step 3: Build and Submit**
```bash
# Build for App Store
eas build --platform ios

# Submit to App Store
eas submit --platform ios
```
